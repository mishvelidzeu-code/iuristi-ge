# ცვლილებების ისტორია

ამ ფაილში აღირიცხება პროექტის ყველა მნიშვნელოვანი ფუნქციური, მონაცემთა ბაზის, დიზაინისა და deployment ცვლილება.

## 2026-07-12

### დამატებულია

- შეიქმნა ქართული სამართლებრივი ტესტების მრავალგვერდიანი responsive პლატფორმა;
- დაემატა Supabase Auth, PostgreSQL სქემა, RLS და უსაფრთხო RPC ფუნქციები;
- დაემატა ტესტის მდგომარეობის `localStorage`-ში შენარჩუნება და რეალურ დროზე დაფუძნებული ტაიმერი;
- შეიქმნა პროფილი, რეიტინგი, ადმინისტრატორის სივრცე და სერტიფიკატის შემოწმება;
- შეიქმნა GitHub repository და ჩაირთო GitHub Pages;
- გამოსწორდა desktop/mobile header-ის არევა და desktop-ზე მობილური მენიუს ღილაკი;
- დაემატა კანონების მიხედვით ტესტების არქიტექტურა: `laws`, `law_articles`, `question_law_articles`;
- Supabase-ში ჩაიტვირთა 7 ძირითადი სამართლებრივი აქტი;
- დაემატა კანონების საჯარო კატალოგი და კონკრეტული კანონის გვერდი;
- შეიქმნა `PROJECT_OVERVIEW.md` და მუდმივი ცვლილებების ჟურნალი.

### გამოსწორდა

- Supabase-ის სერტიფიკატის კოდის PostgreSQL 17 თავსებადობა;
- დაუდასტურებელი ელფოსტის შესვლის პრობლემა და ქართული შეცდომის შეტყობინება;
- ადმინისტრატორის ანგარიშის როლი და ელფოსტის დადასტურება.
